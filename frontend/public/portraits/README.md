# Portrait Images

Place portrait images for each target in this directory. The files should match the names configured in `backend/config.py`.

## Required Images:

### Politicians:
- `lawrence_wong.png` - Lawrence Wong (Prime Minister)
- `lee_hsien_loong.png` - Lee Hsien Loong (Senior Minister)
- `tharman_shanmugaratnam.png` - Tharman Shanmugaratnam (President)
- `k_shanmugam.png` - K. Shanmugam (Minister for Home Affairs)
- `heng_swee_keat.png` - Heng Swee Keat (Deputy Prime Minister)

### Celebrities:
- `jj_lin.png` - JJ Lin (Singer-Songwriter)
- `stefanie_sun.png` - Stefanie Sun (Singer)
- `fandi_ahmad.png` - Fandi Ahmad (Football Legend)

## Image Guidelines:
- **Format**: PNG (recommended) or JPG
- **Size**: Square aspect ratio (e.g., 400x400px or 512x512px)
- **Content**: Professional headshot/portrait photos
- **Quality**: High resolution for best display

## Notes:
- If an image fails to load, the UI will fall back to displaying an icon
- Images are served directly from this directory via the frontend's public folder
